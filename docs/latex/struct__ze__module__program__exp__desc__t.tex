\doxysubsection{Detailed Description}
Module extended descriptor to support multiple input modules. 


\begin{DoxyItemize}
\item Implementation must support \+::\+ZE\+\_\+experimental\+\_\+module\+\_\+program extension
\item Modules support import and export linkage for functions and global variables.
\item SPIR-\/V import and export linkage types are used. See SPIR-\/V specification for linkage details.
\item p\+Input\+Modules, p\+Build\+Flags, and p\+Constants from \+::ze\+\_\+module\+\_\+desc\+\_\+t is ignored.
\item Format in \+::ze\+\_\+module\+\_\+desc\+\_\+t needs to be set to \+::\+ZE\+\_\+\+MODULE\+\_\+\+FORMAT\+\_\+\+IL\+\_\+\+SPIRV.
