Detailed Description

Implementations of the HIP API functions using the CHIP interface providing basic functionality such hipMemcpy, host and device function registration, hipLaunchByPtr, etc. These functions operate on base CHIP class pointers allowing for backend selection at runtime and backend-specific implementations are done by inheriting from base CHIP classes and overriding virtual member functions.
