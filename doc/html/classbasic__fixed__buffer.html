Detailed Description

template<typename Char>
class basic_fixed_buffer< Char >

\rst A fixed-size memory buffer. For a dynamically growing buffer use :class:fmt::basic_memory_buffer.

Trying to increase the buffer size past the initial capacity will throw std::runtime_error. \endrst
