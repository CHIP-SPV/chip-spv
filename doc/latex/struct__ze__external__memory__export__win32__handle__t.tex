Exports an allocation as a Win32 handle.

\item This structure may be passed to \+::ze\+Mem\+Get\+Alloc\+Properties, via the {\ttfamily p\+Next} member of \+::ze\+\_\+memory\+\_\+allocation\+\_\+properties\+\_\+t, to export a memory allocation as a file descriptor.

\item The requested memory export type must have been specified when the allocation was made.
