Contains kernel arguments and a queue on which to execute. Prior to kernel launch, the arguments are setup via \mbox{\hyperlink{class_h_i_pxx_backend_ada55e19c0eaa86cafd4b1d4e2c1758b9}{HIPxx\+Backend\+::configure\+Call()}}. Because of this, we get the kernel last so the kernel so the \mbox{\hyperlink{class_h_i_pxx_exec_item_acdd07ff5a00214faee448682984f23a2}{launch()}} takes a kernel argument as opposed to queue receiving a \mbox{\hyperlink{class_h_i_pxx_exec_item}{HIPxx\+Exec\+Item}} containing the kernel and arguments.
